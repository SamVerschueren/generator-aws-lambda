# generator-aws-lambda

> Scaffold out an AWS Lambda microservice with [bragg](https://github.com/SamVerschueren/bragg).


## Installation

```
$ npm install -g generator-aws-lambda
```


## Usage

With [Yeoman](http://yeoman.io/).

```
$ yo aws-lambda
```
